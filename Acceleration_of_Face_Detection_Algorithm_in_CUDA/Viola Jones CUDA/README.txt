* Make sure nvcc and OpenCV is installed in the testing machine

* Rename the test video to test.avi

* Change following preprocessor directives in main.cpp
	VIDWIDTH to video resolution width
	VIDHEIGHT to video resolution height 

* In the terminal enter following commands to compile and run

	make 		: To compile all
	make run 	: To compile and run
	make clean 	: To clean built files and outputs

* Output video will be output.avi
